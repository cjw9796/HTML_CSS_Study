날로 먹는 코딩은 IT 교육 사이트입니다. 다양한 날로 먹는 IT 관련 강의가 존재합니다.

날로 먹는 코딩의 특징

날로 먹는 코딩은 다른 사이트와 차원이 다른 깔끔한 구성을 보여줍니다.

폭 넓은 강사진

날로 먹는 코딩은 정말 뛰어난 능력을 가진 강사진으로 뽑습니다.

핵심적인 강의

강의는 실제 개발에 있어서 반드시 필요한 핵심적인 내용을 중심으로 다룹니다.

최신강의 목록

C언어 기초 프로그래밍 강의

C언어 강의는 기초 프로그래밍 강의입니다. 처음 프로그램을 접하는 입문자가 듣기 적합한 강의로 강의료는 무료이며 C언어 기초 프로그래밍 강의는 총 20강의로 구성됩니다.

JAVA기초 프로그래밍 강의

JAVA언어 강의는 객체 지향 프로그래밍의 기본을 알려주는 강의 입니다. 처음 프로그램을 접하는 입문자가 듣기 적합한 강의로 강의료는 무료이며 JAVA기초 프로그래밍 강의는 총 30강의로 구성됩니다.

Android 언어 프로그래밍 강의

Android 강의는 java 또는 kotiln을 활용하여 Android 어플리케이션을 개발하는 프로그래밍 강의입니다.
